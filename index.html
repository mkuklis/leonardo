Core

leonardo(...)

Creates Leonardo object. Leonardo object contains a reference to canvas element. Canvas element will be created or referenced from DOM based on given paramenters during Leonardo initialization. All calls to drawing methods depend on Leonardo object so intialization has to happen first.

Parameter types

container String DOM element or the ID of the canvas
or
x Number x coordinate
y Number y coordinate
width Number canvas width
height Number canvas height

Usage 

// creates new leonardo object with 200px x 300px canvas at 50, 100
var leo = leonardo(50, 100, 200, 300);

// creates new leonardo object with a reference to canvas with id = #paper
var leo = leonardo("#paper");

// or if you are a fun of new
var leo = new Leonardo("#paper");

leonardo.circle(x, y, r, [attrs])

Draws a circle.

Parameters

x Number x coordinate of the centre
y Number y coordinate of the centre
r Number radius
attrs Object attributes

Usage 

// creates and draws circle with radius 25 filled with color
var circle = leo.circle(30, 30, 25, {fill: '#FF7400'});

Example

leonardo.rect(x, y, w, h, [r], [attrs])

Draws a rectangle.

Parameters

x Number x coordinate of the centre
y Number y coordinate of the centre
w Number width
h Number height
r Number border radius
attrs Object attributes

Usage 

// creates and draws rectangle filled with color
var rect = leo.rect(25, 30, 40, 30, 0, {fill: '#247bc7'});

leonardo.path(command, [options])

Creates and draws path element based on given command(s).

Parameters

command Object single path command
options Object options
OR
commands Array array of multiple path commands
options Object options

Single path command is a JSON object where the key represents the type of the command and the value represents command's argument(s).

Available Commands

M {M: [x, y, ...]} move to point with x, y coordinates
L {L: [x, y, ...]} draw line to x, y
V {V: x} draw vertical line to x
H {H: y} draw horizontal line to y
Q {Q: [cp1x, cp1y, x1, y1, cp2x, cp2y, x2, y2, ...]} draw quadratric curve
B {B: [cp1x, cp1y, x1, y1, cp2x, cp2y, x2, y2, ...]} draw bezier curve

Usage 

// draw quadratic curve
leo.path([
  {M: [200, 30]},
  {Q: [300, 0, 288, 120]}]);

// draw heart
leo.path([
  {M:[75, 40]},
  {B:[75,37,70,25,50,25, 20,25,20,62.5,20,62.5,
      20,80,40,102,75,120, 110,102,130,80,130,62.5,
      130,62.5,130,25,100,25, 85,25,75,37,75,40]}]);

Example

leonardo.image(src, x, y, w, h, [attrs])

Imports image into canvas.

Parameters

src String image source (URI)
x Number x coordinate
y Number y coordinate
w Number width
h Number height
attrs Object attributes

Usage 

// load image into canvas
var image = leo.image("image.png", 10, 10, 90, 90);
